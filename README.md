# 🛡️ Safety Index Melbourne

A web application that provides safety insights for Melbourne suburbs.  
The project connects to a MySQL database through a JavaScript backend and uses **AJAX** on the frontend for dynamic data fetching.

---

## 🚀 Features
- 📊 **Interactive Safety Index**: Display safety-related statistics for different Melbourne areas  
- 🔄 **AJAX-based API Calls**: Seamless frontend-backend communication without page reloads  
- 🗄️ **MySQL Database**: Store and query suburb-level safety data  
- 🌐 **Responsive Web UI**: Clean, user-friendly interface  

---

## 🛠️ Tech Stack
- **Frontend**: HTML, CSS, JavaScript, AJAX  
- **Backend**: Node.js / Express.js (JavaScript)  
- **Database**: MySQL  
- **Hosting**: GitHub Pages (frontend), backend can be deployed on any Node.js-supported server
